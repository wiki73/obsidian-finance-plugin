# Obsidian Finance Tracker

Плагин для Obsidian, который помогает вести учёт финансов по месяцам с удобной визуализацией через Canvas.

## ✨ Возможности

- Учёт доходов и расходов через форму
- Разделение данных по месяцам (отдельные файлы)
- Автоматическое создание файлов диаграмм
- Интеграция с Obsidian Canvas
- Поддержка пользовательских категорий
- Валюта: рубли (₽)

## 📁 Структура файлов
```/Finance/
    /Months/
        2025-01.md
    /Charts/
        2025-01-expenses-by-category.md
        2025-01-expenses-percent.md
        2025-01-income.md
/Settings/
    finance-settings.json```


## 🚀 Установка плагина

1. Скачайте или склонируйте репозиторий
2. Скопируйте папку плагина в:
   `Vault/.obsidian/plugins/`
3. Включите плагин в:
   `Settings → Community plugins`

## 🛠 Использование

Через палитру команд (`Ctrl + P`):
- "Добавить расход"
- "Добавить доход"

После ввода данные автоматически сохраняются в файл месяца и обновляют диаграммы.

## 🔧 Настройки

Вы можете указать:
- Путь к Canvas-файлу
- Пользовательские категории
- Валюту

## 📌 Планы развития

- История по месяцам
- Аналитика расходов
- Экспорт в CSV
